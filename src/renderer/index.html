Как добавить свои игры

          

            Приложение загружает игры с основного сервера автоматически.
            Дополнительно вы можете положить свои ROM-файлы в локальную папку
            games/SMD/ рядом с приложением — они появятся в каталоге
            с пометкой Свои.
          


          
Структура папки игры

          
games/
  SMD/
    Название игры/          ← папка с именем игры
      cover.jpg             ← обложка (рекомендуется 800×1200 px, формат 2:3)
      info.data             ← описание (первая строка = заголовок)
      ИмяИгры.gen           ← ROM-файл Sega Mega Drive (обязательно)
      cheats.md             ← чит-коды в формате Markdown (необязательно)
      logo.png              ← логотип для героя-экрана (необязательно)
      видео.mp4             ← видео-прохождение (необязательно)
      screens/              ← папка со скриншотами
        screen1.png
        screen2.jpg
        ...
      OST/                  ← папка с музыкой из игры
        01. Track.mp3
        02. Track.mp3
        ...


          
Поддерживаемые форматы

          

            
              
Тип файлаФорматы

            
            
              
ROM.gen

              
Обложка / скриншоты.png, .jpg, .jpeg, .webp

Формат файла info.data

          
Contra: Hard Corps

Легендарный run-and-gun от Konami. Выберите одного из четырёх
бойцов и уничтожьте армию мутантов.

• Жанр: Action / Run and Gun
• Игроки: 2
• Год: 1994
• Разработчик: Konami
• Рекомендованные игры: Gunstar Heroes, Battletoads and Double Dragon

          
    
            
  • Первая строка — название игры (отображается как заголовок карточки).
    • 
            
  • Строки, начинающиеся с •, разбираются как метаданные.
    • 
            
  • Поля Жанр и Игроки используются в фильтрах.
    • 
            
  • Поле Рекомендованные игры строит блок похожих игр.
    • 
          


          
Требования к обложке

          
    
            
  • Рекомендуемое соотношение сторон: 2:3 (ширина:высота).
    • 
            
  • Минимальная высота: 1200 px.
    • 
            
  • При отсутствии обложки приложение покажет название игры вместо изображения.
    • 
          


          
Примечания

          
    
            
  • Локальные игры отображаются в каталоге с пометкой Свои.
    • 
            
  • Если в локальной папке есть игра с тем же именем, что и на сервере — приоритет у локальной версии.
    • 
            
  • Оценки и «Буду играть» сохраняются локально; личный рейтинг также отправляется на общий сервер для формирования средней оценки.
