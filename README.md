# RETRO SMD Emulator

Десктопный лаунчер и эмулятор игр Sega Mega Drive (`.gen`).

Игры загружаются с удалённого сервера автоматически. Дополнительно можно положить свои ROM-файлы в локальную папку `games/SMD/` — они появятся в каталоге с пометкой **«Игра пользователя»**.

---

## Возможности

- Каталог игр с обложками, описанием, скриншотами, видео-прохождением и OST
- Эмулятор Sega Mega Drive (EmulatorJS) прямо внутри приложения
- Серверная библиотека + поддержка локальных пользовательских игр
- Оценки (1–5 ★): личная сохраняется локально, средняя — агрегируется на сервере
- Фильтры: жанр, количество игроков, первая буква, сортировка
- Режимы «Мои игры» и «Буду играть»
- Полное управление геймпадом: Xbox / PS4 (DualShock 4) / PS5 (DualSense)
- Индикатор подключённого геймпада в топбаре
- Документация по добавлению своих игр прямо внутри приложения

---

## Быстрый старт

```bash
# Установить зависимости и запустить
npm install
npm start
```

Или двойной клик по **`start-retro.bat`** — откроется меню:

```
[1] Запустить приложение
[2] Загрузить / обновить игры из \games на сервер
[3] Выход
```

## Добавить свои игры

1. Создайте папку `games/SMD/<Название игры>/` рядом с приложением.
2. Положите туда файлы по структуре:

```
games/
  SMD/
    Название игры/
      cover.jpg             ← обложка (рекомендуется 800×1200, соотношение 2:3)
      info.data             ← описание (первая строка = заголовок)
      ИмяИгры.gen           ← ROM-файл (обязательно)
      cheats.md             ← чит-коды в Markdown (необязательно)
      logo.png              ← логотип для hero-экрана (необязательно)
      видео.mp4             ← видео-прохождение (необязательно)
      screens/              ← скриншоты (.png, .jpg, .webp)
      OST/                  ← музыка (.mp3, .ogg, .flac и др.)
```

3. Перезапустите приложение — игра появится с пометкой «Игра пользователя».

Подробная инструкция: **Документация → Добавить свои игры** внутри приложения.

## Управление геймпадом

| Кнопка | Каталог | Страница игры |
|---|---|---|
| **A / Cross** | Открыть игру | Играть |
| **B / Circle** | — | Назад |
| **X / Square** | «Буду играть» | «Буду играть» |
| **Y / Triangle** | Фильтры вкл/выкл | Таб «Чит-коды» |
| **LB / L1** | — | Таб «Скриншоты» |
| **RB / R1** | — | Таб «Видео» |
| **Select / Share** | Фильтры вкл/выкл | Назад |
| **Start / Options** | «Мои игры» вкл/выкл | Играть |
| **DPad / Левый стик** | Навигация | Прокрутка страницы |
| **Правый стик** | — | Прокрутка страницы |

## Обновление библиотеки на сервере

1. Скопируйте `upload.cfg.example` → `upload.cfg` и укажите данные сервера.
2. Запустите `start-retro.bat` → **[2] Загрузить / обновить игры**.

Скрипт синхронизирует `games/SMD/` с сервером:
- Загружает только новые и изменённые файлы
- Для `.data` / `.md` — проверяет MD5, чтобы поймать правки текста
- Видеофайлы (`.mp4` и др.) не загружаются — остаются только локально

## Серверная часть

Исходники сервера — в папке `server/`.

```bash
cd server
npm install
node server.js          # или: pm2 start server.js --name retro-api
```

API:
- `GET  /api/games`   — список всех игр
- `GET  /api/ratings` — средние оценки сообщества
- `POST /api/ratings` — отправить оценку `{ gameId, value }`
- `POST /api/reload`  — сбросить кэш игр (вызывается автоматически после upload)
